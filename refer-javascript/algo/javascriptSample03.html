1부터 N까지 합 출력하기

Q. 자연수 N이 입력되면 1부터 N까지의 합을 출력하는 프로그램을 작성하시오.

입력 설명
20이하의 자연수 N이 입력됩니다.
출력 설명
1부터 N까지의 합이 출력됩니다.
입력 에제
10
출력 예제
55

var num = 10;
function solution(num){
    let sum = 0;
    for(let i = 1; i <=num ; i++){
        sum += i;
    }
    return sum;
}
solution(num);
