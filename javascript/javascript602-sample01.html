CSS 속성 검색하기

  • align-content 속성은 콘텐츠의 상하관계 정렬 상태를 정의합니다.
  • align-items 속성은 콘텐츠 내부의 정렬 상태를 정의합니다.
  • align-self 속성은 콘텐츠의 정렬 상태를 정의합니다.
  • all 속성은 요소의 속성을 초기화 또는 상속을 설정합니다.
  • animation 속성은 애니메이션을 정의합니다.
  • appearance 속성은 운영 체제의 테마를 기반으로 플랫폼 기본 스타일을 사용하여 요소를 표시하는데 사용됩니다.
  • aspect-ratio 속성은 상자에 대한 기본 가로 세로 비율을 설정합니다.
  • break-after 속성은 상자가 생성된 이후 페이지, 열 또는 영역 구분의 동작 방식을 설정합니다
  • break-before 속성은 상자가 생성된 이전에 페이지, 열 또는 영역 구분의 동작 방식을 설정합니다.
  • break-inside 속성은 생성된 상자 내에서 페이지, 열 또는 영역 구분의 동작 방식을 설정합니다.
  • color-scheme 속성은 요소가 편안하게 렌더링할 수 있는 색 구성표를 나타낼 수 있습니다.
  • contain 속성은 요소와 그 내용이 문서 트리의 나머지 부분과 독립적이라는 것을 나타낼 수 있습니다.
  • content-visibility 속성은 렌더링 작업을 잠재적으로 생략할 수 있습니다.
  • counter-set 속성은 CSS 카운터를 지정된 값으로 설정합니다.
  • font-feature-settings 속성은 OpenType 글꼴의 고급 타이포그래픽 기능을 제어합니다.
  • font-kerning 속성은 커닝 기능을 사용할 것인지의 여부를 지정하는 속성입니다.
  • font-language-override 속성은 lang 속성으로 정의된 언어를 다른 언어로 대체하는 속성입니다.
  • font-optical-sizing 속성은 텍스트 렌더링이 다른 크기로 볼 수 있도록 최적화되었는지 여부를 설정합니다.

  • font-synthesis 속성은브라우저에서 굵게 또는 기울임꼴로 합성할 수 있는 누락된 서체를 제어합니다.
  • font-variant-alternates 속성은 대체 글리프의 사용을 제어합니다.
  • font-variant-caps 속성은 대문자로 대체 글리프를 사용할 수 있습니다.
  • font-variant-east-asian 속성은 동아시아 스크립트에 대체 글리프를 사용하는 것을 제어합니다.
  • font-variant-ligatures 속성은 적용되는 요소의 텍스트 내용에 사용되는 연결 및 상황별 형식을 제어합니다.
  • font-variant-numeric 속성은 숫자, 분수 및 순서 표시자에 대한 대체 글리프의 사용을 제어합니다.
  • font-variant-position 속성은 대첨자 또는 첨자로 배치된 대체 작은 글리프의 사용을 제어합니다.
  • font-variation-settings 속성은 에서는 변경할 특성의 네 글자 축 이름과 해당 값을 지정합니다.
  • forced-color-adjust CSS 속성은 작성자가 특정 요소를 강제 색상 모드에서 제외할 수 있습니다.
  • gap 속성은 행과 열 사이의 간격을 설정합니다.
  • grid-template-areas 속성은 명명된 그리드 영역을 지정하여 그리드에 셀을 설정하고 이름을 할당합니다.
  • image-rendering 속성은 이미지 스케일링 알고리즘을 설정합니다.
  • inline-size 속성은 요소 블록의 쓰기 모드에 따라 수평 또는 수직 크기를 정의합니다.
  • justify-items 속성은 상자의 모든 항목에 대해 기본 just-self를 정의하여 적절한 축을 따라 각 상자를 정당화하는 기본 방법을 제공합니다.
  • justify-self 속성은 선형 컨테이너 내부에서 정렬되는 방식을 설정할 수 있습니다.
  • line-break 속성은 구두점 및 기호를 사용할 때 중국어, 일본어 또는 한국어텍스트의 줄을 끊는 방법을 설정합니다.
  • line-height-step 속성은 선 상자 높이에 대한 계단 단위를 설정합니다
  • mask-border 속성은 CSS 단축기 속성을 사용하여 요소의 테두리를 따라 마스크를 만들 수 있습니다.
  • mask-type 속성은 SVG 마스크 요소를 휘도 또는 알파 마스크로 사용할지 여부를 설정합니다.
  • mask 속성은 속성은 특정 지점에서 이미지를 마스킹하거나 클리핑하여 요소(부분 또는 전체)를 숨깁니다.
  • orphans 속성은 페이지, 영역 또는 열의 맨 아래에 표시해야 하는 블록 컨테이너의 최소 행 수를 설정합니다
  • overflow-anchor 속성은 브라우저의 스크롤 고정 동작을 해제할 수 있습니다.
  • overflow-block 속성은 상자의 블록 시작 및 블록 끝 모서리에 오버플로되는 내용을 표시합니다.
  • overflow-clip-margin 속성은오버플로우가 있는 요소의 범위를 얼마나 벗어나는지 결정합니다.
  • overflow-inline 속성은 콘텐츠가 상자의 인라인 시작 및 끝 모서리에 오버플로되는 경우를 보여 주는 항목을 설정합니다.
  • overflow-wrap 속성은 텍스트가 줄 상자에 넘치지 않도록 해야 하는지 여부를 설정합니다.
  • overscroll-behavior-block 속성은 스크롤 영역의 블록 방향 경계에 도달했을 때 브라우저의 동작을 설정합니다.
  • overscroll-behavior-inline 속성은 동작 인라인 CSS 속성은 스크롤 영역의 인라인 방향 경계에 도달했을 때 브라우저의 동작을 설정합니다.
  • overscroll-behavior-x 속성은 스크롤 영역의 수평 경계에 도달했을 때 브라우저의 동작을 설정합니다.
  • overscroll-behavior-y 속성은 스크롤 영역의 수직 경계에 도달했을 때 브라우저의 동작을 설정합니다.
  • overscroll-behavior 속성은 스크롤 영역의 경계에 도달할 때 브라우저가 수행하는 작업을 설정합니다.
  • place-content 속성은 블록 방향과 인라인 방향을 동시에 조정할 수 있습니다.
  • place-items 속성은 블록 방향과 인라인 방향을 따라 항목을 한 번에 정렬할 수 있습니다.
  • place-self 속성은 개별 항목을 블록 방향과 인라인 방향으로 동시에 정렬할 수 있습니다.
  • rotate 속성은 transforms 속성과 독립적으로 회전 변환을 지정할 수 있습니다.
  • row-gap 속성은 요소의 그리드 행 사이의 간격 크기를 설정합니다.
  • ruby-align 속성은 서로 다른 루비 요소의 베이스에 대한 분포를 정의합니다.
  • ruby-position 속성은 기본 요소에 상대적인 루비 요소의 위치를 정의합니다.
  • scale 속성은 척도 변환 특성과 독립적으로 척도 변환을 지정할 수 있습니다.
  • scroll-behavior 속성은 스크롤 API에 의해 스크롤이 트리거될 때 스크롤 상자에 대한 동작을 설정합니다.
  • scroll-margin-bottom 속성은 스냅하는 데 사용되는 스크롤 스냅 영역의 아래쪽 여백을 정의합니다.
  • scroll-margin-inline-start 속성은 인라인 차원의 시작 부분에 있는 스크롤 스냅 영역의 여백을 정의합니다.
  • scroll-margin-inline 속성은 인라인 치수에 있는 요소의 스크롤 여백을 설정합니다.
  • scroll-margin-left 속성은 스크롤 스냅 영역의 왼쪽 여백을 정의합니다.
  • scroll-margin-right 속성은 스크롤 스냅 영역의 오른쪽 여백을 정의합니다.
  • scroll-margin-top 속성은 스크롤 스냅 영역의 윗쪽 여백을 정의합니다.
  • scroll-padding-block-end 속성은 최적 보기 영역의 블록 치수에 있는 끝 모서리에 대한 간격띄우기를 정의합니다.
  • scroll-padding-block-start 속성은 최적 보기 영역의 블록 치수에 있는 시작 모서리에 대한 오프셋을 정의합니다.
  • scroll-padding-bottom 속성은 스크롤 포트의 최적 보기 영역 하단에 대한 오프셋을 정의합니다.
  • scroll-padding-inline-end 속성은 최적 보기 영역의 인라인 치수에 있는 끝 모서리에 대한 오프셋을 정의합니다.
  • scroll-padding-inline-start 속성은 최적 보기 영역의 인라인 치수에 있는 시작 모서리에 대한 오프셋을 정의합니다.
  • scroll-padding-inline-bottom 속성은 최적 보기 영역의 인라인 치수에 있는 하단 모서리에 대한 오프셋을 정의합니다
  • scroll-padding-inline-end 속성은 스크롤 포트의 최적 보기 영역의 인라인 치수에 있는 끝 모서리에 대한 오프셋을 정의합니다.
  • scroll-padding-inline-start 최적 보기 영역의 인라인 치수에 있는 시작 모서리에 대한 오프셋을 정의합니다.
  • scroll-padding-inline 속성은 인라인 치수에 있는 요소의 스크롤 패딩(scroll padding)을 설정합니다.
  • scroll-padding-left 속성은 스크롤 포트의 최적 보기 영역 왼쪽에 대한 오프셋을 정의합니다.
  • scroll-padding 속성은 요소의 모든 측면에 한 번에 스크롤 패딩을 설정합니다.
  • scroll-snap-type 속성은 스크롤 컨테이너에 스냅 지점이 있는 경우 스냅 지점을 얼마나 엄격하게 적용하는지를 설정합니다.
  • scrollbar-color 속성은 스크롤 막대 트랙과 엄지손가락의 색상을 설정합니다.
  • scrollbar-width 속성은 스크롤 막대가 표시될 때 최대 두께를 설정할 수 있습니다.
  • shape-image-threshold 속성은 형상을 추출하는 데 사용되는 알파 채널 임계값을 설정합니다.
  • shape-margin 속성은 쉐이프 외부를 사용하여 작성된 CSS 쉐이프의 여백을 설정합니다.
  • shape-outside 속성은 인접한 인라인 콘텐츠를 감싸야 하는 쉐이프를 정의합니다.
  • text-emphasis-color 속성은 강조 표시의 색상을 설정합니다
  • text-emphasis-position 속성은 강조 표시가 그려지는 세트입니다.
  • text-emphasis-style 속성은 강조 표시의 모양을 설정합니다.
  • text-emphasis 속성은 텍스트에 강조 표시를 적용합니다(공백 및 컨트롤 문자 제외).
  • text-orientation 속성은 한 줄에 있는 텍스트 문자의 방향을 설정합니다.
  • text-rendering 속성은 텍스트를 렌더링할 때 최적화할 대상에 대한 정보를 렌더링 엔진에 제공합니다.
  • text-underline-position 속성은 텍스트 장식 속성의 밑줄 값을 사용하여 설정할 밑줄의 위치를 지정합니다.
  • touch-action 속성은 터치스크린 사용자가 요소의 영역을 조작하는 방법을 설정합니다
  • translate 속성은 transforms 속성과 독립적으로 변환 변환을 지정할 수 있습니다.
  • widows 속성은 표시해야 하는 블록 컨테이너의 최소 행 수를 설정합니다.
  • will-change 속성은 브라우저에 요소의 변경 방법을 알려주는 속성 힌트입니다.
  • writing-mode 속성은 텍스트 행을 가로 또는 세로로 배치할지 여부와 블록이 진행되는 방향을 설정합니다.
